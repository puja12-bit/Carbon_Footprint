# MASTER PROMPT

Build a production-quality full-stack application called **Carbon Copilot AI**.

## Core Vision

Carbon Copilot is an intelligent assistant that estimates and explains the carbon footprint of user actions **before they happen**, instead of tracking them afterward.

Whenever a user is:

* typing a shopping query
* ordering food
* booking flights
* booking hotels
* ordering cabs
* selecting delivery options
* planning travel
* choosing transportation
* buying electronics
* scheduling meetings
* selecting cloud services
* streaming media
* making any digital decision

the system should instantly estimate:

* Estimated CO₂ emissions
* Why that amount occurs
* Greener alternatives
* Money savings
* Carbon savings
* Confidence score

The assistant should feel like GitHub Copilot but for sustainability.
